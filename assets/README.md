# Assets Directory

This directory contains images and media files for the Modern Card Collection project.

## Files to Add:

### Social Media Images (for SEO)
- `og-image.jpg` - Open Graph image (1200x630px) for Facebook/LinkedIn sharing
- `twitter-card.jpg` - Twitter Card image (1200x600px) for Twitter sharing
- `preview.png` - Main preview image for README

### Favicons
- `favicon.ico` - Standard favicon
- `favicon-32x32.png` - 32x32 PNG favicon
- `favicon-16x16.png` - 16x16 PNG favicon  
- `apple-touch-icon.png` - Apple touch icon (180x180px)

### Optional
- Any additional images for card examples
- Icons or graphics used in the components

## Image Specifications

### Open Graph Image (og-image.jpg)
- Size: 1200x630 pixels
- Format: JPG (optimized for web)
- Content: Screenshot of the card collection with title overlay

### Twitter Card Image (twitter-card.jpg)  
- Size: 1200x600 pixels
- Format: JPG (optimized for web)
- Content: Similar to OG image but optimized for Twitter's format

### Preview Image (preview.png)
- Size: 1200x800 pixels (or larger)
- Format: PNG for crisp screenshots
- Content: Full page screenshot or collage of the best cards

### Favicons
- Use a simple, recognizable icon (maybe a card shape or stylized 'C')
- Ensure good visibility at small sizes
- Consistent with the overall design aesthetic

## Tools for Creating Images

- **Screenshots**: Use browser dev tools or tools like [shots.so](https://shots.so)
- **Favicons**: Use [favicon.io](https://favicon.io) or [realfavicongenerator.net](https://realfavicongenerator.net)
- **Design**: Figma, Canva, or Photoshop for creating branded images
